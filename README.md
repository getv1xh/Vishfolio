## Overview

Vishfolio is a smooth, modern, and visually restrained portfolio experience focused on:

* premium UI composition
* cinematic motion
* elegant typography
* subtle depth
* interactive presentation

The goal was to create a portfolio that feels calm, polished, and memorable without relying on excessive effects or visual noise.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/vishfolio.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Design Direction

Vishfolio focuses on:

* restraint over excess
* atmosphere over noise
* motion with intention
* elegant spacing
* clean interaction systems

The visual identity uses:

* matte black surfaces
* soft cyan gradients
